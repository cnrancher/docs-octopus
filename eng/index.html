Octopus

Lightweight edge device management system based on Kubernetes

Get Started
StarFork

Build for Edge

Lightweight and does not need to replace any components of the Kubernetes clusters. With Octopus deployed, the cluster can have the ability to manage any edge device as a resource.

Simple yet Flexible

Octopus has build-in device adaptor like BLE, modbus, OPC-UA, and MQTT, and it supports custom device adaptor plugins. Go ahead and build your own adaptor.

Powered by Kubernetes & k3s

Both ARM64 and ARMv7 are supported with multiarch images. Works great with both valina Kubernetes and k3s.
